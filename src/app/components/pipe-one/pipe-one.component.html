با pipe ها میتونیم نحوه نمایش یک مقدار رو تو صفحه تمپلیت تغییر بدیم. تو مثال های پایین کاربرد برخی از پایپ هارو نشون دادیم

{{ ' 1600000 '}} با پایپ number میشه: {{ ' 1600000 ' | number}}
{{ ' 12.638467846 ' }} با پایپ number:2.2-5 میشه: {{ ' 12.638467846 ' | number:'2.2-5' }}
{{ ' 0.7 ' }} با پایپ percent میشه: {{ ' 0.7 ' | percent }}
{{ date }} با پایپ date میشه: {{ date | date }}
{{ date }} با پایپ date:shortDate میشه: {{ date | date:'shortDate' }}
{{ date }} با پایپ date:shortTime میشه: {{ date | date:'shortTime' }}
{{ ' this is title ' }} با پایپ titlecase میشه: {{ ' this is title ' | titlecase }}

یک مقدار json یا یک object رو میشه با پایپ json نمایش داد:

{{ json | json }}

اگه پایپ json رو نزاریم به صورت [object,object] نشون میده
